# 寓境 MVP 验收清单

## 1. 首页空态
- [ ] 无故事时显示空态文案「输入一个概念，成为第一个解锁者」

## 2. 路径 B — 新生成
- [ ] 输入新概念 → SSE/打字机 → 解锁 → unlock_count +1

## 3. 路径 B — 缓存复用
- [ ] 第二用户同概念 → 不重复调 LLM（from_cache）→ 打字机播放

## 4. 并发等待
- [ ] 生成进行中第二用户 → 等待页 skeleton → poll 成功进入阅读

## 5. 低置信度确认
- [ ] confidence < 0.85 → 确认页 → 开始/重新输入

## 6. 登录与藏书阁
- [ ] 解锁后「收入藏书阁」→ 微信登录 → 我的页可见
- [ ] 藏书阁重读为 full 模式

## 7. 分享
- [ ] onShareAppMessage 仅含故事标题，不含概念名
- [ ] 分享落地新用户走 blind 流程

## 8. Admin
- [ ] /admin 列表可见 concept_name
- [ ] 重生后 unlock_count 保留

## 9. 失败处理
- [ ] 生成失败 → 错误页 + 重试，锁已释放
